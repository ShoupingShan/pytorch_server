 <div class="lemma-summary" label-module="lemmaSummary">
  <div class="para" label-module="para">
   水晶饼是陕西省
   <a href="/item/%E6%B8%AD%E5%8D%97%E5%B8%82" target="_blank">
    渭南市
   </a>
   <a data-lemmaid="10763101" href="/item/%E4%B8%8B%E9%82%BD%E9%95%87/10763101" target="_blank">
    下邽镇
   </a>
   的地方名点。相传，它是宋代当地民众赞美
   <a data-lemmaid="220184" href="/item/%E5%AF%87%E5%87%86/220184" target="_blank">
    寇准
   </a>
   而制作的糕点。通过制皮、制馅、制酥、包酥、成型的烹饪步骤完成。
  </div>
  <div class="para" label-module="para">
   水晶饼金面银帮，
   <a data-lemmaid="3424720" href="/item/%E8%B5%B7%E7%9A%AE/3424720" target="_blank">
    起皮
   </a>
   掉酥，口感油多而不腻，糖重而渗甜，具有浓郁的
   <a href="/item/%E7%8E%AB%E7%91%B0" target="_blank">
    玫瑰
   </a>
   芳香。新中国成立后，水晶饼多次被国家商业部、陕西省商业厅授予优质产品称号。
  </div>

 <div class="para-title level-3" label-module="para-title">
  <h3 class="title-text">
   <span class="title-prefix">
    水晶饼
   </span>
   起源
  </h3>
 </div>
 <div class="para" label-module="para">
  相传
  <a href="/item/%E5%8C%97%E5%AE%8B%E6%97%B6%E6%9C%9F" target="_blank">
   北宋时期
  </a>
  ，有一年
  <a href="/item/%E5%AF%87%E5%87%86" target="_blank">
   寇准
  </a>
  从京都开封回到故乡下邽县 (今渭南市
  <a href="/item/%E4%B8%8B%E9%82%BD%E9%95%87" target="_blank">
   下邽镇
  </a>
  )探亲。适逢其50大寿，乡党们为表特意送来寿桃、面花、寿匾，表示祝贺，寇准特设宴款待。
 </div>
 <div class="para" label-module="para">
  <a class="lemma-album layout-right nslog:10000206" href="/pic/%E6%B0%B4%E6%99%B6%E9%A5%BC/2853744/18852120/1f178a82b9014a9021f65587ae773912b21beec4?fr=lemma&amp;ct=cover" nslog-type="10000206" style="width:137px;" target="_blank" title="寇准">

   </div>
   <div class="description">
    寇准

 </div>
 <div class="para" label-module="para">
  在酒过三巡，手下人捧进精致的桐木盒子，
  <a href="/item/%E5%AF%87%E5%87%86" target="_blank">
   寇准
  </a>
  打开一看，里面装着50个晶莹透亮如同水晶石一般的点心。在点心上面，还放着一张红纸，整整齐齐地写着一首诗：“公有水晶目，又有水晶心，能辨忠与奸，清白不染尘。”落款是渭北老叟。后来，寇准的家厨也仿照做出了这种点心，寇准据其特点给它取了一个好听的名字，叫做“水晶饼”。

 </div>
 <div class="para" label-module="para">
  元代时，水晶饼已远销京、津各大城市。
 </div>
 <div class="para" label-module="para">

 <div class="para-title level-2" label-module="para-title">
  <h2 class="title-text">
   <span class="title-prefix">
    水晶饼
   </span>
   制作方法
  </h2>
  <a class="edit-icon j-edit-link" data-edit-dl="3" href="javascript:;">
  </a>
 </div>
 <div class="para" label-module="para">
  1.制皮：将
  <a href="/item/%E5%8A%A8%E7%89%A9%E6%B2%B9" target="_blank">
   动物油
  </a>
  加5公斤
  <a href="/item/%E6%B0%B4" target="_blank">
   水
  </a>
  在和面机内搅拌，油、水溶合均匀后加富强粉搅拌，待面团光滑、离手为宜。
 </div>
 <div class="para" label-module="para">
  2.制馅：将全部白砂糖、猪板油、熟板油、熟面粉，经加工后的各种辅料，混合搅匀即成馅。
 </div>
 <div class="para" label-module="para">
  3.制酥：用富强粉总数的32%、动物油总数的50%混合搅拌均匀。
 </div>
 <div class="para" label-module="para">
  4.包酥：按50%的皮，42%的酥的比例，包合一起，压平，擀成圆形，由中心向外卷成条，切块待用。
 </div>
 <div class="para" label-module="para">
  5.成型：用制好的包酥小块拍成直径5厘米，厚2.8厘米的正圆形饼，上盖红色“水晶饼”印章，待烘烤。
 </div>

  1.将细砂糖，小麦淀粉，糯米粉过筛到一个大容器里，用筷子搅拌均匀。
 </div>
 <div class="para" label-module="para">
  2.然后加入1奶粉勺食用油，略为搅拌。
 </div>
 <div class="para" label-module="para">
  3.222克清水用电热水壶煮沸腾，然后慢慢加入到粉里。
 </div>
 <div class="para" label-module="para">
  4.用筷子搅拌成絮状。
 </div>
 <div class="para" label-module="para">
  5.然后用手揉成面团。
 </div>
 <div class="para" label-module="para">
  6.取一小撮，揉圆，压扁。
 </div>
 <div class="para" label-module="para">
  7.放入豆沙馅料。
 </div>
 <div class="para" label-module="para">
  8.包裹好后，稍微压扁，并在表面抹上一层色拉油。(也可以印上一层小麦淀粉代替抹油，目的是防粘。)
 </div>
 <div class="para" label-module="para">
  9.印油的一面朝内，将其放入月饼模内。
 </div>
 <div class="para" label-module="para">
  10.在吸油纸上压出饼模。(如果饼坯粘住模具难以脱模，就用刀刮下来吧。)
 </div>
 <div class="para" label-module="para">
  11.重复上诉步骤，一共做了13个水晶饼。
 </div>
 <div class="para" label-module="para">
  12.蒸锅里的水沸腾后，放入饼坯，大火蒸15分钟，然后熄火闷5分钟即可。
 </div>

 <div class="para-title level-3" label-module="para-title">
  <h3 class="title-text">
   <span class="title-prefix">
    水晶饼
   </span>
   营养价值
  </h3>
 </div>
 <div class="para" label-module="para">
  <b>
   原料价值
  </b>
 </div>
 <div class="para" label-module="para">
  精制猪板油，
  <a href="/item/%E7%8C%AA%E6%B2%B9" target="_blank">
   猪油
  </a>
  尤其猪板油中含有多种脂肪酸，饱和脂肪酸和不饱和脂肪酸的含量相当，几乎平分秋色，具有一定的营养，并且能提供极高的热量。在人体的消化吸收率较高，可达95%以上，是维生素A和维生素D含量很高的调料，所含的脂肪比例小于黄油，较适于缺乏维生素A的人和少年儿童。
 </div>
 <div class="para" label-module="para">
  <a href="/item/%E9%9D%92%E7%BA%A2%E4%B8%9D" target="_blank">
   青红丝
  </a>
  ，其第一核心材料
  <a href="/item/%E6%A1%94%E5%AD%90%E7%9A%AE" target="_blank">
   桔子皮
  </a>
  营养丰富，色香味兼优，广受消费者的青睐，一方面桔子皮中含有大量的维生素C和香精油，能够起到提神、通气的作用，而且气味清香。第二方面桔子皮具有
  <a href="/item/%E7%90%86%E6%B0%94%E5%8C%96%E7%97%B0" target="_blank">
   理气化痰
  </a>
  、健胃除湿、降低血压等功能。
 </div>
 <div class="para" label-module="para">
  青红丝中的另一材料
  <a href="/item/%E8%90%9D%E5%8D%9C" target="_blank">
   萝卜
  </a>
  味甘、辛、性凉，入肺、胃、肺、大肠经，具有清热生津、凉血止血、下气宽中、消食化滞、开胃健脾、顺气化痰的功效，可用于腹胀停食、腹痛、咳嗽、痰多等症；
 </div>
 <div class="para" label-module="para">
  青红丝中的第三个主材
  <a data-lemmaid="176" href="/item/%E6%9C%A8%E7%93%9C/176" target="_blank">
   木瓜
  </a>
  性温、味酸，入肝、脾经; 具有消食，驱虫，清热，祛风的功效; 主治胃痛，消化不良，肺热干咳，乳汁不通，湿疹，寄生虫病，手脚痉挛疼痛等病症。
 </div>

   禁忌
  </h3>

 <div class="para" label-module="para">
  动物油热量高、
  <a href="/item/%E8%83%86%E5%9B%BA%E9%86%87%E9%AB%98" target="_blank">
   胆固醇高
  </a>
  ，故老年人、肥胖和
  <a href="/item/%E5%BF%83%E8%84%91%E8%A1%80%E7%AE%A1%E7%97%85" target="_blank">
   心脑血管病
  </a>
  患者都不宜食用。一般人食用动物油也不要过量。很多奶油制品含有大量的糖，所以也不宜吃得太多，糖尿病患者也应注意。
